python 定义变量

标识符（变量、模块名、函数名、类名等自己命名的）

规则

  • 有数字、字母、下划线组成，不可以数字开头
  • 标识符对大小写敏感。
  • 不能使用关键字

规范

  • 见名知意
  • 大驼峰 or 小驼峰 or 下划线_
  • 变量、函数、模块名使用下划线连接
  • 类名大驼峰
