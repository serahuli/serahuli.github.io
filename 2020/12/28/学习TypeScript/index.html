学习TypeScript

微软开发，开源，是js的超集，会被编译为js代码

  • Typescript 特点

    • 始于js，归于js
    • 强大的类型系统
    • 先进的js

  • TypeScript 安装

    检查： tsc -V

  • TypeScript编译

    在html直接引入ts文件，浏览器会报错（如果ts文件中只有js代码，可以正常使用）；如果包括ts代码，需要编译成js文件后才可以使用， ts中 let -> js var

  • vscode自动编译
